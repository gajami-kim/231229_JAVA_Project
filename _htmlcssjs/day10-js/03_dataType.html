변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number => 정수, 싱수, NaN(not a Number : 숫자가 아님)
  • 문자 : String => "", '', `` 감싼 값들
  • boolean : true / false
  • 객체 : object => 1개 이상의 데이터가 특정 구조를 가지고 있는 형태, 배열과 혼합해서 많이 사용됨 {키 : 값} {} : 아무것도 없는 사실도 1개의 객체로 취급
  • 배열 : Array => [1,2,3,4] 형태의 집합, 데이터타입이 같지 않아도 됨 (java의 list와 비슷 / 순서와 길이(length)가 존재)
  • 알수없는 값 : null => 인지가 안됨
  • 미지정된 값 : undefined => 값이 할당되지 않음
  1. 기본자료형 : 값이 메모리의 주소값을 가르키는 곳에 직접 할당이 되어있는 값 num, boolean, string(참조자료형이지만 편의상 기본자료형과 같이 사용)
  2. 참조자료형 : 값이 저장되어 있는 주소를 할당 string, arr, object, funtion, null, undefined ...
  3. == 같다 (값만 비교) / != 같지않다 === 같다(값과 타입을 비교) !== 같지않다(값과 타입을 비교)
