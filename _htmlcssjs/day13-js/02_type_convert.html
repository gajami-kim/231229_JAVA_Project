타입변환 casting(형변환)

  • 문자 -> 숫자 : Number('string'), parseInt('1'), parseFloat('0.5');
  • 숫자-> 문자 : 123.toString()
  • 문자 -> boolean : Boolean('true')
  • boolean -> 문자 : String(true)
  • 숫자 -> boolean : Boolean(1) / Boolean(0) => 보통 1은 true, 0은 false
  • boolean -> 숫자 : Number(true), Number(false) => 1,0 출력
